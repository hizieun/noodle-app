# Release Notes

## v1.0.0 (Initial Release)

**출시일**: 2026년 03월 06일

Noodle 노포 맛집 탐험기 초기 버전 런칭! 기존 Python 맛집 크롤러 스크립트를 기반으로, 모던하고 아름다운 React 웹 애플리케이션을 성공적으로 구축했습니다.

### ✨ 새로운 기능 (New Features)
- **프론트엔드 React 앱 초기화**: Vite와 React를 사용하여 빠른 속도의 SPA 구축.
- **다크 모드 & 글래스모피즘 UI**: 고급스러운 어두운 배경과 반투명한 컴포넌트, 부드러운 호버 애니메이션으로 프리미엄 식당 탐색 경험 제공.
- **스마트 이모지 포맷팅**: 상호명에 포함된 키워드(예: 냉면, 횟집, 고기, 포차)를 분석하여 알맞은 음식 이모지를 자동으로 제목 앞에 부착하는 매칭 로직 개발.
- **깔끔한 드롭다운 지역 필터링**: 지역구를 한눈에 알아보기 쉽도록 가나다 순으로 정렬된 `<select>` 드롭다운 제공 (전체보기 고정).
- **자동 데이터 변환 스크립트 결합**: 기존 `맛집_평점순_정렬.csv` 파일을 React에서 바로 읽을 수 있는 `data.json`으로 변환하는 자동화 스크립트 작성 완료.

### 🌐 배포 (Deployment)
- **Vercel 프로덕션 배포 완료**: 언제든지 외부에서 접근 가능한 Public URL 구성.

### 🛠 수정 및 개선 사항 (Fixes & Refactoring)
- 기본 템플릿(JS 대신 TSX)으로 인한 화면 미출력 렌더링 에러 해결. (전체 로직을 안정적인 확장자 체계에 맞게 수정)
- 무분별한 카카오맵 알파벳 라벨(A, B, C...) 제거 로직 추가.
